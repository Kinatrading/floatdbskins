# floatdbskins

Chrome-розширення для формування посилань на CSFloat DB:
- вибір колекції зі списку (дані з `skins.json`);
- налаштування **Special**: `Normal` (`category=1`), `StatTrak™` (`category=2`), `Souvenir` (`category=3`) або без категорії за замовчуванням;
- налаштування **Rarity**: `Any` (без параметра) або `1..7`;
- автоматичний обхід rarity `1..6` (без `Contraband=7`) з відкриттям кожної сторінки окремо, зчитуванням `Items Found` та підсумком по кожній рідкості;
- якщо з'являється помилка `there are no items of that rarity in the selected collection - 4`, рідкість пропускається як порожня.
- перед автозбором парситься `skins.json`, визначається тип (`collection`/`crate`) і доступні rarity для вибраного пункту;
- до кожної rarity додається шанс випадіння (для collection: 80% зі спадом ×5, для crate: 79.92/15.98/3.2/0.64/0.26);
- опція додаткової перевірки `StatTrak` (`category=2`) для підсумку normal + stattrak.
- встановлення діапазону float через подвійний повзунок (крок `0.01`) або ручне введення;
- ID колекції нормалізується з нижнім підкресленням (наприклад, `set_dust_2_2021`).

## Локальний запуск
1. Відкрийте `chrome://extensions`.
2. Увімкніть **Режим розробника**.
3. Натисніть **Load unpacked** та виберіть корінь цього репозиторію.
4. Натисніть на іконку розширення — інтерфейс відкриється в **окремій вкладці**.
