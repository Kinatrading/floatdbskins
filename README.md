# floatdbskins

Chrome-розширення для швидкого формування посилання на CSFloat DB:
- вибір колекції зі списку (дані з `skins.json`);
- встановлення діапазону float через подвійний повзунок (крок `0.01`) або ручне введення;
- генерація URL формату `https://csfloat.com/db?min=0&max=1&collection=set_realism_camo`.

## Локальний запуск
1. Відкрийте `chrome://extensions`.
2. Увімкніть **Режим розробника**.
3. Натисніть **Load unpacked** та виберіть корінь цього репозиторію.
